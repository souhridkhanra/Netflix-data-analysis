🎬 Netflix Data Analysis
📌 Project Overview

This project explores and analyzes Netflix’s dataset to uncover insights about content distribution, trends, and user engagement patterns. The analysis focuses on understanding the growth of Netflix content over the years, genre popularity, and differences between movies and TV shows.

🎯 Objectives
Analyze the distribution of Movies vs TV Shows on Netflix
Identify the most common genres and categories
Study content addition trends over time
Explore country-wise content production
Gain insights into Netflix’s content strategy
📊 Dataset

The dataset includes information such as:

Title
Type (Movie / TV Show)
Director & Cast
Country
Date Added
Release Year
Genre (Listed In)
Description
🛠️ Tools & Technologies Used
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
🔍 Key Insights
Netflix has significantly increased its content library in recent years
Movies dominate the platform compared to TV shows
Certain countries contribute a major share of content
Drama, Comedy, and Documentary are among the most frequent genres
Content additions peaked in specific recent years, showing rapid expansion

🚀 How to Run the Project

Clone the repository:

git clone https://github.com/souhridkhanra/Netflix-data-analysis.git

Navigate to the project folder:

cd Netflix-data-analysis

Install required libraries:

pip install -r requirements.txt

Open Jupyter Notebook:

jupyter notebook
